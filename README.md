Bring the color of the following Xcode themes to Visual Studio Code:

-   Xcode Default (Dark)
-   Xcode Default Customized Version (Dark)
-   Xcode Classic (Dark)
-   Xcode Classic (Light)
-   Xcode Civic (Dark)
-   Xcode Low Key (Light)
-   Xcode Spartan (Light)

Matches the default window and syntax colors of Xcode.
